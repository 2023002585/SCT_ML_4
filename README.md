# SCT_ML_4

Hand Gesture Recognition using Support Vector Machine (SVM)

## Project Overview
This project develops a hand gesture recognition model that identifies and classifies different hand gestures from image data. The model is trained using a subset of the LeapGestRecog dataset and implemented using a Support Vector Machine (SVM) classifier.

## Dataset
LeapGestRecog Hand Gesture Dataset

## Technologies Used
- Python
- NumPy
- OpenCV
- Matplotlib
- Scikit-learn

## Methodology
1. Loaded gesture images from the dataset.
2. Converted images to grayscale.
3. Resized images to 64×64 pixels.
4. Flattened images into feature vectors.
5. Split data into training and testing sets.
6. Trained a Linear SVM classifier.
7. Evaluated performance using accuracy score.

## Results
- Model Accuracy: **99%**
- Successfully classified multiple hand gesture categories.
- Demonstrated high recognition performance on test data.

## Conclusion
The SVM-based hand gesture recognition system achieved excellent classification accuracy and can be used as a foundation for gesture-based human-computer interaction applications.
